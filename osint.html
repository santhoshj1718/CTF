Introduction to OSINT

Open Source Intelligence (OSINT) involves collecting information from publicly available sources. In CTFs, OSINT challenges typically involve:

  • Finding hidden information about people or organizations
  • Tracking digital footprints across platforms
  • Analyzing metadata from files and images
  • Discovering relationships between entities
  • Identifying locations from photos or clues

Common OSINT Challenge Types

Type Description Example Techniques
Username Tracking Find accounts across platforms Username search tools, profile analysis
Email Investigation Find accounts and breaches Email search, breach databases
Domain Research Discover website information WHOIS, DNS records, subdomains
Image Analysis Extract metadata and locations EXIF data, reverse image search
Geolocation Identify locations from clues Google Earth, street view, landmarks

OSINT Methodology

1. Define the target (username, email, image, etc.) 2. Identify potential sources of information 3. Use appropriate tools for investigation 4. Correlate information from multiple sources 5. Verify findings for accuracy 6. Document the process and sources
Note: Always respect privacy laws and terms of service when conducting OSINT research.

Profile Analysis

1. Check profile pictures (reverse image search) 2. Analyze bio/description information 3. Look for connected accounts/links 4. Examine post history for clues 5. Note registration dates and activity patterns 6. Check for common friends/connections

# Install git clone https://github.com/sherlock-project/sherlock.git cd sherlock python3 -m pip install -r requirements.txt # Usage python3 sherlock username1 username2 python3 sherlock --csv --output results.csv username

Maigret - Advanced username search

# Install pip install maigret # Usage maigret username maigret -a username # Advanced mode maigret --ids username # Show site IDs
CTF Tip: For username challenges:
  • Try variations of the username (add numbers, underscores)
  • Check archived profiles using Wayback Machine
  • Look for GitHub repositories or code snippets
  • Search for email addresses associated with the username
  • Don't ignore smaller or niche platforms

CTF OSINT Tips

General Strategy

  • Start with basic searches (Google, username checkers)
  • Document all findings for correlation
  • Try multiple tools and approaches
  • Look for connections between pieces of information
  • Verify information from multiple sources

Common Flag Locations

# Typical flag formats: flag{Th1s_1s_4_fl4g} FLAG{Th1s_1s_4_fl4g} CTF{Th1s_1s_4_fl4g} # Common hiding places: - Image metadata - Old website versions - Social media bios - GitHub repositories - Domain registration info

Quick Checks

  • Search for usernames across platforms
  • Check WHOIS records for domains
  • Look at image EXIF data
  • Search breach databases for emails
  • Check archived versions of websites
